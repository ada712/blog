    旋转链表

    Ada 2022/11/12

    给你一个链表的头节点 head ，旋转链表，将链表每个节点向右移动 k 个位置。

    示例 1：

    输入：head = [1,2,3,4,5], k = 2 输出：[4,5,1,2,3]

    /**
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**`
 * @param {ListNode} head
 * @param {number} k
 * @return {ListNode}
 */
var rotateRight = function(head, k) {

    //处理边界条件
    if (k === 0 || !head || !head.next) {
        return head;
    }
    // 累计总长度
    let tail = head;curr = head;
    let j=1;
    while(tail.next){
        tail = tail.next;
        j++;
    }
    // 向右移动a个节点
    let a= k%j
    if(a == 0) return head;
    
    let real = j - a;
    while(real >1){
        curr = curr.next;
        real--;
    }
    let newHead = curr.next;
    curr.next = null;
    tail.next = head;
    return newHead;
};
