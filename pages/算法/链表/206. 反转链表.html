    206.反转链表

    Ada 2023/1/10

    给你单链表的头节点 head ，请你反转链表，并返回反转后的链表

    var reverseList = function(head) {
  let prev = new ListNode(-10000,null);
  let curr = head;

  while(curr){
    //当要反转指针时，先记录下原先的指向
    let next = curr.next;
    // 反转
    curr.next = prev;
    // 2指针向前走
    prev = curr;
    curr = next;
  }
  return prev;
};
