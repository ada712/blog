    # 考点是： VUE3 多重筛选

    请你实现以下功能 4个分类按钮展示默认样式 药品 母婴 食品 百货
goods 里有goodsId goodsName goodsType

药品 下有 布洛芬 阿司匹林 感冒颗粒 母婴 奶粉 宝宝面霜 纸尿裤 火腿
婴儿车 食品 好丽友 苹果 方便面 牛奶 百货 沐浴露 牙膏 排插 衣架

点击类型则显示激活且展示对应的货物
再次点击则取消展示对应药物，
要是全部都没有点击则默认展示全部

    分析：
为了实现这个功能，你可以使用 Vue 3 的组合式 API，比如 ref 和 computed。下面是一个基本的实现思路：

1.创建一个 ref 来存储当前激活的分类。
2.使用 computed 来计算应该显示哪些商品。
3.在模板中绑定这个 computed 属性到你的商品列表。
下面是具体的代码实现：

    <template>
  <div>
    <button v-for="type in types" :key="type" @click="toggleType(type)">
      {{ type }}
      <span v-if="isActive(type)">[Active]</span>
    </button>
    <ul>
      <li v-for="good in filteredGoods" :key="good.goodsId">
        {{ good.goodsName }}
      </li>
    </ul>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue';

const types = ['药品', '母婴', '食品', '百货'];
const goods = [
  { goodsId: 1, goodsName: '布洛芬', goodsType: '药品' },
  { goodsId: 2, goodsName: '阿司匹林', goodsType: '药品' },
  { goodsId: 3, goodsName: '感冒颗粒', goodsType: '药品' },
  { goodsId: 4, goodsName: '奶粉', goodsType: '母婴' },
  { goodsId: 5, goodsName: '宝宝面霜', goodsType: '母婴' },
  { goodsId: 6, goodsName: '纸尿裤', goodsType: '母婴' },
  { goodsId: 7, goodsName: '火腿', goodsType: '母婴' },
  { goodsId: 8, goodsName: '婴儿车', goodsType: '母婴' },
  { goodsId: 9, goodsName: '好丽友', goodsType: '食品' },
  { goodsId: 10, goodsName: '苹果', goodsType: '食品' },
  { goodsId: 11, goodsName: '方便面', goodsType: '食品' },
  { goodsId: 12, goodsName: '牛奶', goodsType: '食品' },
  { goodsId: 13, goodsName: '沐浴露', goodsType: '百货' },
  { goodsId: 14, goodsName: '牙膏', goodsType: '百货' },
  { goodsId: 15, goodsName: '排插', goodsType: '百货' },
  { goodsId: 16, goodsName: '衣架', goodsType: '百货' },
];

// 维护一个激活的类型列表
const activeTypes = ref([]);

const toggleType = (type) => {
  if (activeTypes.value.includes(type)) {
    // 如果当前类型已经激活，则取消激活
    activeTypes.value = activeTypes.value.filter((t) => t !== type);
  } else {
    activeTypes.value.push(type);
  }
};

const isActive = (type) => activeTypes.value.includes(type);

const filteredGoods = computed(() => {
  if (activeTypes.value.length === 0) {
    // 如果没有激活的类型，则显示所有商品
    return goods;
  }
  return goods.filter((good) => activeTypes.value.includes(good.goodsType));
});
</script>

    # 考点是： VUE3 数据筛选获取

    提供一个数据列表mock ,搜索框实现模糊查询 ，将选中选项添加进已选择列表 已选择列表中，点击选项的删除按钮，将该选项移除
    1
    为了实现这个需求，我们可以使用 Vue 3 的响应式特性来管理数据和状态。下面是一个简单的实现方案，包括数据列表的模拟、搜索框的模糊查询以及已选择列表的管理。

数据列表模拟
首先，我们创建一个模拟的数据列表，用于演示搜索和选择功能
// mock.js
export const mockData = [
  { id: 1, name: 'Apple' },
  { id: 2, name: 'Banana' },
  { id: 3, name: 'Cherry' },
  { id: 4, name: 'Date' },
  { id: 5, name: 'Elderberry' },
  // 更多数据...
];

接下来，在 Vue 3 组件中，我们将实现搜索框、数据列表和已选择列表的功能。
<template>
  <div>
    <input v-model="searchQuery" placeholder="Search..." />
    <ul>
      <li v-for="item in filteredItems" :key="item.id">
        {{ item.name }}
        <button @click="addToSelected(item)">Select</button>
      </li>
    </ul>
    <h3>Selected Items:</h3>
    <ul>
      <li v-for="selectedItem in selectedItems" :key="selectedItem.id">
        {{ selectedItem.name }}
        <button @click="removeFromSelected(selectedItem)">Remove</button>
      </li>
    </ul>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue';
import { mockData } from './mock.js';

const searchQuery = ref('');
const selectedItems = ref([]);

const filteredItems = computed(() => {
  return mockData.filter(item =>
    item.name.toLowerCase().includes(searchQuery.value.toLowerCase())
  );
});

const addToSelected = (item) => {
  if (!selectedItems.value.some(i => i.id === item.id)) {
    selectedItems.value.push(item);
  }
};

const removeFromSelected = (item) => {
  selectedItems.value = selectedItems.value.filter(i => i.id !== item.id);
};
</script>

    # 考点： VUE3 生命周期钩子

    // useKeyboardVisibilityToggle.js
import { onMounted, onUnmounted, ref } from 'vue';

export function useKeyboardVisibilityToggle() {
  const isVisible = ref(true);

  const toggleVisibility = () => {
    isVisible.value = !isVisible.value;
  };

  const handleKeydown = (event) => {
    if (event.keyCode === 37 || event.keyCode === 38) {
      toggleVisibility();
    }
  };

  onMounted(() => {
    window.addEventListener('keydown', handleKeydown);
  });

  onUnmounted(() => {
    window.removeEventListener('keydown', handleKeydown);
  });

  return { isVisible };
}

使用自定义钩子

<template>
  <div :style="{ display: isVisible ? 'block' : 'none' }">
    <!-- Your component content here -->
  </div>
</template>

<script setup>
import { useKeyboardVisibilityToggle } from './useKeyboardVisibilityToggle';

const { isVisible } = useKeyboardVisibilityToggle();
</script>
